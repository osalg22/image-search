In order to use this, enter a search term in the following format in the address bar:

url..../api/imagesearch/<searchterm>

The result will be an array of objects showing the returned images' url, source url, thumbnail url, and snippet:

[ { "url": "http://www.example.com", "page_url": "http://www.example.com", "thumbnail_url: "http://www.example.com", "snippet": "This is a snippet" } ... ]

You can view 10 results at a time. To view more results you can add an offset in the url:

url...../api/imagesearch/<searchterm>?offset=<number>

You can also find the 10 most recently searched terms by going to:

url..../api/latest/imagesearch

Note: This tool only returns up to 100 results.
